# PassChecker

It is a Password Checker script, it will check and display the count of the given passwords that how many times they are hacked.

What you need?
-Python3

PassChecker_File.py
-In this you have to save passwords (Only one password in a line) in a file called "password.txt" and it will check all the passwords in that file, to check the passwords just open CMD or Terminal and type "python PassChecker_File.py". Make sure yopu have both passchecker script and password.txt in the same space/folder.

PassChecker_Terminal.py
-In this you have to write the passwords in the terminal when you are going to run the script, i.e. "python PassChecker_Terminal.py password123 password456 789456123"
you can type as many passwords as you want.
